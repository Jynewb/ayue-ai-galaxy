💻 小白安装指南（3种方式）

方式一：Mac/Linux 一键安装（推荐）

curl -fsSL https://openclaw.ai/install.sh | bash

方式二：Docker 安装（更简单）

docker run -d -p 8080:8080 \
  -e ANTHROPIC_API_KEY=your_key \
  openclaw/openclaw:latest

方式三：手动部署

git clone https://github.com/openclaw/openclaw.git
cd openclaw
pnpm install
pnpm build
pnpm start
