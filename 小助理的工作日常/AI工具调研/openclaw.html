💻 小白安装指南（3种方式）

方式一：Mac/Linux 一键安装（推荐）

curl -fsSL https://openclaw.ai/install.sh | bash

方式二：Docker 安装（更简单）

docker run -d -p 8080:8080 \
  -e ANTHROPIC_API_KEY=your_key \
  openclaw/openclaw:latest

方式三：手动部署

git clone https://github.com/openclaw/openclaw.git
cd openclaw
pnpm install
pnpm build
pnpm start

🔗 接入飞书/微信/钉钉

1
飞书配置：创建企业应用 → 获取App ID和Secret → 配置权限 → 填写到OpenClaw配置
2
微信配置：注册企业微信 → 创建应用 → 获取API权限
3
钉钉配置：创建钉钉应用 → 开启机器人 → 获取Webhook地址
