## 🚀 使用方法

### 第一步：获取飞书密钥

1. 打开 [飞书开放平台](https://open.feishu.cn/)
2. 创建「企业自建应用」
3. 获取 **App ID** 和 **App Secret**
4. 添加权限：
   - `im:message:send_as_bot`（发送消息）
   - `im:message:receive_as_bot`（接收消息）

### 第二步：运行脚本

```bash
# 下载脚本
curl -O https://your-website.com/openclaw-install.sh

# 运行
bash openclaw-install.sh
```

### 第三步：输入飞书密钥

脚本运行后，输入您的 App ID 和 App Secret 即可！
