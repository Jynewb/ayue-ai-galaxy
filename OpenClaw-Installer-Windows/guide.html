🚀 OpenClaw

Windows 安装配置指南

版本 5.0 | 作者：TAO小助理

1 运行安装脚本

1.1 双击运行脚本

解压安装包后，双击运行 Install.bat

1.2 自动检查环境

脚本会自动检查并安装：Node.js、pnpm、OpenClaw

1.3 选择 AI 模型

输入数字选择模型（已测试可用）：
- 1. MiniMax M2.5 (Standard)
- 2. MiniMax M2.5 HighSpeed
- 3. Qwen3.5 (Free)
- 4. Kimi K2.5
- 5. DeepSeek V3
- 6. GPT-4o
- 7. Claude 4
- 8. Gemini 2.5 Pro
- 9. GLM-5

1.4 输入 API Key

输入对应模型的 API Key（如已测试通过的 DeepSeek）

1.5 输入飞书 App ID

这里先留空，我们先到飞书开放者后台

2 创建飞书应用

2.1 创建企业自建应用

1. 打开 https://open.feishu.cn/
2. 点击「创建企业自建应用」
3. 填写应用名称（如「我的AI助手」）
4. 点击「创建」

2.2 添加机器人

1. 点击左侧「添加应用能力」
2. 点击「按能力添加」
3. 找到「机器人」卡片，点击「添加」→ 确认

2.3 获取凭证

1. 点击左侧「凭证与基础信息」
2. 复制「App ID」和「App Secret」

3 开通权限

3.1 批量导入权限

1. 点击左侧「权限管理」
2. 点击右上角「批量导入/导出权限」
3. 粘贴以下 JSON： 
{
  "scopes": {
    "tenant": [
      "im:message",
      "im:message.p2p_msg:readonly",
      "im:message.group_at_msg:readonly",
      "im:message:send_as_bot",
      "contact:contact.base:readonly",
      "contact:user.base:readonly"
    ]
  }
}
4. 点击「下一步」→「确认新增权限」→「申请开通」

3.2 发布应用

1. 点击左侧「版本管理与发布」
2. 点击「创建版本」
3. 版本号填 1.0.0
4. 点击「保存」→「发布」

4 继续配置

4.1 回到命令窗口

此时安装脚本会等待，输入之前复制的：
1. 飞书 App ID
2. 飞书 App Secret
等待自动配置完成

4.2 配置长连接

1. 回到飞书后台，点击「事件与回调」
2. 「订阅方式」选择「长连接」
3. 点击「保存」（此时 Gateway 需保持运行）

4.3 添加接收消息事件

1. 点击「添加事件」
2. 搜索 im.message.receive_v1
3. 勾选后点击「确定」

4.4 重新发布

点击「版本管理与发布」，再次点击「发布」

5 授权与启动

5.1 飞书对话

1. 在飞书和机器人对话
2. 它会回复你一个飞书匹配 ID

5.2 飞书授权

1. 新打开一个命令提示符
2. 将机器人回复的那段授权命令复制进去
3. 执行授权

5.3 重启 Gateway

1. 关闭所有命令窗口
2. 重新打开命令提示符
3. 输入 openclaw gateway 启动
🎉 配置完成！
现在可以在飞书发消息和 AI 对话了！

祝你使用愉快！🚀
