📋 工具简介

OpenClaw是开源智能体框架，2026年3月以24.8万GitHub星标超越Linux成为全球最受欢迎开源项目。它创造了一种全新的模型消费场景，让AI从"一问一答"变成自动化执行。

🔥 2026年3月最新更新（3.1版本）

  • 智能体推理默认优化：不同模型有不同脾气，现在可以精细化配置
  • 企业级运行架构：支持Docker、Kubernetes部署，自动扩缩容
  • Android节点：手机也能成为AI节点
  • 多平台联动：微信、钉钉、飞书、Siri四大入口
  • 8000+开箱即用Skill：普通用户也能轻松使用

📖 使用教程

1. 安装

npm install openclaw
openclaw init my-agent

2. 配置模型

# config.yaml
model: qwen-plus
temperature: 0.8
max_tokens: 2000

3. 创建智能体

openclaw create agent my-assistant
